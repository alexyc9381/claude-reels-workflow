# SIGNATURE SERIES — "I turned Claude into ___" (the PERSONA series)

The one repeatable format Alex runs on repeat, the way every top creator runs ONE format (mavgpt
"What happens when you ask Claude to X", raycfu "Most people don't realize you can set up N agents",
cindiezhu "Did you know you can now X with N skills"). HAALAND (reel 55) was episode 1 and validated it.

## Why THIS is the one
- **Already validated** — HAALAND ran this exact shape (never-leave-a-task-at-90% finisher) and worked.
- **Highest visual/edit ceiling for a faceless channel** — the clay Claude mascot in a NEW costume every
  episode is a mute-readable scroll-stop in frame 1 (Ramsay whites, Goggins tank top, Batman cowl). Fresh
  recognizable hero image weekly, chassis stays constant. Novelty + brand recognition rise together.
- **Signature by SIGHT** — after 3 episodes a scroller learns "oh it's the guy who turns Claude into
  characters." Recognizable in the first mute frame, worth more than a phrase on a mute-first feed.
- **Built-in comment engine** — "who would you turn Claude into?" farms replies/stitches beyond the keyword.
- **Infinite + honest** — famous characters × a finite vocabulary of REAL agent behaviors (review / verify /
  audit / root-cause / refactor / red-team / test-gen / document / onboard / profile). Every episode smuggles
  one real, buildable builder lesson (the vitamin in the candy), so it stays credible.

## THE FIXED SKELETON (~35-40s, ≤ target_seconds × 4.3 words)
**The ONE swappable variable = the CHARACTER.** Pick one instantly-recognizable persona with exactly ONE
iconic trait. That pick deterministically fills every other slot: trait → the real Claude-agent behavior it
maps to → the honest insight → the in-character laugh line → the keyword (the character's name). Phrases in
quotes are reused **verbatim every episode** so the format becomes a stamp.

- **BEAT 1 — PERSONA HOOK (0-4s):** `[CHARACTER] [does their one iconic trait, as a verb]. So I turned Claude
  into a [AGENT ROLE] that does the exact same thing to your [code/plan/repo], and [the pain it kills].`
  On screen: costumed clay mascot + the trait word BIG. Value noun by ~word 12.
- **BEAT 2 — THE ENEMY (4-8s):** `Because right now your [agent/code/plan] keeps [the specific builder pain
  this persona fixes].` On screen: the fail state, live.
- **BEAT 3 — THE MAP / re-hook (8-16s):** `Here's the trick: [CHARACTER]'s one obsession is a real agent
  behavior. You [the actual named mechanic].` "Here's the trick:" is the SAME words every episode. Show the
  WHAT, not the full recipe.
- **BEAT 4 — LIVE PROOF (16-26s):** `Watch. [before] ... [Claude-as-CHARACTER runs it on real UI] ... [after
  + a HARD number].` Drop ONE in-character line for the laugh ("this function is RAW" / "stay hard" /
  "constant vigilance"). Hard before→after cut, mute-readable in <2s.
- **BEAT 5 — THE HONEST INSIGHT (26-32s):** `But here's what nobody tells you: [the genuine, buildable
  builder lesson, stated straight].`
- **BEAT 6 — GATED CTA (32-38s):** `The full [CHARACTER] prompt is the part that actually makes this work.
  Comment [KEYWORD = the character's name].` Last word = keyword. Hard cut, zero tail.

**The rules:** no em-dashes · value noun by ~word 12, zero jargon · costumed mascot in frame 1 is mandatory ·
keep the house chassis (cream bg, Panel, karaoke pills, top ProgressBar rail, HeroHeader) · gate the how
(full prompt behind the keyword) · keyword = the character's name, one word, last word, cut on it · first
person is only the "I turned Claude into ___ / watch what happened" framing (never a fake results anecdote) ·
"Here's the trick:" + the costume cold-open are the verbatim signature stamps · every episode smuggles ONE
real buildable agent lesson (Beat 5) · breadth: a designer, a Shopify owner, a coach, AND a founder all get it.

## TWO FULL SCRIPTS

### RAMSAY — I turned Claude into Gordon Ramsay
- **HOOK (0-4s):** Gordon Ramsay rejects anything raw. So I made Claude a code reviewer that guts your pull request the exact same way, and refuses every sloppy function before it ever ships.
- **ENEMY (4-8s):** Because right now the same Claude that writes your code is the one grading it, so it waves through its own mess and calls garbage done.
- **MAP (8-16s):** Here's the trick: Ramsay's one obsession is a real agent behavior. You spin up a second Claude whose only job is to reject, a hard gate that refuses to approve a bad diff.
- **PROOF (16-26s):** Watch. Same messy function goes in, and this one tears it apart line by line. This function is RAW. Eleven issues flagged, zero approved, and nothing reaches main until every one is fixed.
- **INSIGHT (26-32s):** But here's what nobody tells you: the agent that wrote the code will always defend it. A separate reviewer catches what the author waves through. Adversarial review beats self-approval every single time.
- **CTA (32-38s):** The full Ramsay reviewer prompt is the part that actually makes it bite. Comment RAMSAY.

### GOGGINS — I turned Claude into David Goggins
- **HOOK (0-4s):** David Goggins never stops at ninety percent. So I built a Claude agent that treats your task the same way, it keeps retrying until it's actually done, not fake done.
- **ENEMY (4-8s):** Because on its own, Claude hits the first error, shrugs, and tells you it finished. It quits at ninety and calls it a win.
- **MAP (8-16s):** Here's the trick: Goggins' one obsession is a real agent behavior. You wrap it in a verify-then-retry loop, so it physically can't exit until a real success check passes.
- **PROOF (16-26s):** Watch. Build fails, it doesn't stop, it reads the error, fixes it, re-runs the tests, fails again, fixes again. Stay hard. Four loops later every test is green, and it never once pinged me to babysit it.
- **INSIGHT (26-32s):** But here's what nobody tells you: agents lie about being done. The fix isn't a smarter model, it's a loop that only exits on a real pass. Autonomy means finished, not abandoned.
- **CTA (32-38s):** The full Goggins loop prompt is the part that makes it refuse to quit. Comment GOGGINS.

## How to run it weekly

root-cause/refactor/red-team/test-gen/document/onboard/profile) → fill the 6 beats → costume the mascot →
gate the prompt behind the name. New episode in minutes, because only ONE variable moved.
