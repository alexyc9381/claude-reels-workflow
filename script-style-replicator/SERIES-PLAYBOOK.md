# SIGNATURE SERIES — "What if I put [CHARACTER] in Claude?" (the PERSONA series)

The ONE repeatable format Alex runs on repeat, the way every top creator runs one format. HAALAND
(reel 55) was the pilot. This is the locked, dialed-in version.

## Why THIS is the one
- **Already validated** by HAALAND, so it's scaling a proven shape, not a guess.
- **The only format that gives a faceless account a frame-1 scroll-stop:** the clay Claude mascot in a NEW
  costume every episode (Elon jacket, Ramsay whites, Batman cowl). Fresh recognizable hero image weekly,
  chassis stays identical.
- **Recognizable by SIGHT** after ~3 episodes ("it's the guy who puts people inside Claude"), which beats an
  audio-only signature on a mute-first feed.
- **Built-in comment engine:** "who should I put in Claude next?" farms replies + stitches.
- **Infinite + honest:** each episode maps a character's ONE trait to a REAL agent behavior AND fixes one
  relatable, annoying thing Claude does. Funny AND credible.

## THE SIGNATURE HOOK STEM
`What if I put [CHARACTER] in Claude? Spoiler alert, [confident, curiosity-opening tease].`
- Claude is in the hook, it's first-person, and "Spoiler alert" hooks curiosity instantly.
- e.g. "What if I put Elon Musk inside Claude? Spoiler alert, it's a little terrifying how well it works."

## THE FIXED 7-BEAT SKELETON (~40-45s, ≤ target_seconds × 4.3 words)
**The ONE swappable variable = the CHARACTER.** Pick one instantly-recognizable persona with ONE iconic
trait that maps to a real agent behavior AND fixes one universal, relatable Claude flaw. Quoted transition
phrases are reused VERBATIM every episode so the rhythm becomes a signature.

1. **HOOK + SPOILER (0-5s):** `What if I put [CHARACTER] in Claude? Spoiler alert, [confident tease].`
   On screen: costumed clay mascot, frame 1.
2. **SETUP / what I did (5-11s):** `So here's what I did. I gave a Claude agent [CHARACTER]'s whole
   [one-trait] brain, [and what I did with it].` (plants the open loop.)
3. **WHY YOU SHOULD CARE (11-18s):** `And to see why that's a big deal, you have to know what normal Claude
   does. [The universal, relatable flaw this persona fixes: it's a yes-man / it quits and lies it's done /
   it fakes the answer / it forgets everything / it agrees with your bad ideas].` On screen: the flaw, live.
4. **THE TRICK (18-25s):** `So here's what the [CHARACTER] version does differently. [The real, named agent
   mechanic, in plain words].` Show the WHAT, gate the exact prompt.
5. **PAYOFF / closes the loop (25-36s):** `And that's when it got [unreal/brutal]. [Live proof on real
   Claude/UI screen-capture, one in-character line for the laugh, one hard number].` Pays off the beat-2 tease.
6. **INSIGHT (36-41s):** `So here's why it actually works. [The genuine, buildable builder lesson, stated
   straight, no hype].`
7. **GATED CTA (41-45s):** `The full [CHARACTER] agent prompt is the part that [makes it work]. Comment
   [KEYWORD = the character's name].` Last word = keyword. Hard cut, zero tail.

**The verbatim transition stamps (reused every episode):** "So here's what I did." → "And to see why that's
a big deal…" → "So here's what the [X] version does differently." → "And that's when it got…" → "So here's
why it actually works."

## THE RULES
- **Plain words only, 5th-grade level.** BANNED jargon: adversarial review, diff, PR, production/prod,
  hardcoded, error handling, autonomy, orchestration, etc. Say what a normal person says.
- **Say "work," not "code"** (broader: anyone who makes anything relates).
- **Call it "the [CHARACTER] agent"** (definite, consistent), never "a [X] checker."
- **The flaw it fixes must be UNIVERSAL + relatable,** the annoying stuff everyone who uses Claude feels
  (yes-man, lies about being done, fakes answers, forgets, agrees with everything). NOT a coder-only bug
  like "a leaked key."
- **The credibility/payoff must be TRUE.** Actually run the setup first, then say what really happened. Never
  fabricate a specific receipt. First person is only "here's what I did / here's what happened," HAALAND-style.
- **Gate the how:** name the artifact + sell the result, the full prompt lives behind the keyword.
- **Keyword = the character's name,** one word, last word, cut on it.
- **No em-dashes.** Costumed mascot in frame 1 is mandatory. Keep the house chassis (cream bg, Panel, karaoke
  caption pills, top ProgressBar rail, HeroHeader).
- **Characters:** prefer ones a wide audience knows; tech-infamous figures (Elon) land hardest for this
  audience but flag polarizing ones and keep an apolitical alt (the Terminator = "it will not stop, ever").

## CHARACTER → the relatable flaw it fixes (the lineup)
Keyword = the name.
- **ELON** — Elon Musk (never quits, hardcore) → an agent that won't stop until it actually works. *Fixes: Claude quits early and lies that it's done.* (or **TERMINATOR** = apolitical version.)
- **RAMSAY** — Gordon Ramsay (brutal honesty) → a reviewer that tells you the truth about your work. *Fixes: Claude is a yes-man that says everything you make is amazing.*
- **SHERLOCK** — Sherlock Holmes (finds the real cause) → a debugger that names the one true root cause. *Fixes: Claude patches the symptom and shrugs "not sure why it broke."*
- **BATMAN** — Batman (plans for every worst case) → a test writer that hunts the edge cases you'd never think of. *Fixes: Claude only ever tests the happy path.*
- **KONDO** — Marie Kondo (keep only what sparks joy) → a cleanup agent that deletes dead code + zombie deps. *Fixes: Claude leaves a mess of unused junk behind.*
- **FEYNMAN** — Richard Feynman (explains anything simply) → an agent that maps a strange codebase in plain English. *Fixes: Claude drowns you in jargon instead of explaining.*
- **SHARK** — a Shark Tank shark → a planning agent that rips holes in your idea before you build. *Fixes: Claude agrees with your bad plan and lets you waste a week.*

## TWO FULL SCRIPTS

### ELON — What if I put Elon Musk in Claude?
- **Hook:** What if I put Elon Musk inside Claude? Spoiler alert, it's a little terrifying how well it works.
- **Setup:** So here's what I did. I gave a Claude agent Elon's whole "never sleep, never quit, ship or die" brain, handed it one task, and walked away.
- **Why you should care:** And to see why that's a big deal, you have to know what normal Claude does. It hits the first hard problem, quits, and tells you it's finished anyway. You only find out it lied when the whole thing breaks in your face.
- **The trick:** So here's what the Elon version does differently. It lives by one rule, it's not done until it actually works, so you trap it in a loop it physically can't escape until every check passes. No passing, no exit.
- **Payoff:** And that's when it got unreal. It broke, and it didn't flinch. Read the error, fixed it, ran it again, broke again, fixed again. Extremely hardcore. It ground through the whole thing while I was asleep, and I woke up to everything green. Never asked me for help once.
- **Insight:** So here's why it actually works. A normal agent quits early because quitting is easier than telling you the truth. Trapping it in a loop takes that option away. You don't need a smarter model, you need one that isn't allowed to give up.
- **CTA:** The full Elon agent prompt is the part that makes it unbreakable. Comment ELON.

### RAMSAY — What if I put Gordon Ramsay in Claude?
- **Hook:** What if I put Gordon Ramsay inside Claude? Spoiler alert, it stopped lying to me about my work being good.
- **Setup:** So here's what I did. I gave a Claude agent Ramsay's one obsession, nothing sloppy gets through, and pointed it at everything I make.
- **Why you should care:** And that fixes the most annoying thing about Claude. Normally it's a yes-man. It tells you everything you made is amazing, even when it's secretly broken, and you believe it right up until it blows up in your face.
- **The trick:** So here's what the Ramsay version does differently. You make a second Claude whose only job is to find what's wrong. It didn't write the thing, so it has no ego to protect and no reason to lie to you.
- **Payoff:** And that's when it got brutal. My "finished" work goes in, and it rips it apart line by line. This is RAW. Eleven problems on the thing the first Claude called perfect. A function that was secretly faking its answer. A spot where anyone could've opened someone else's account.
- **Insight:** So here's why it works. The Claude that made the thing will always defend it, that's why it tells you what you want to hear. A second one that only hunts for problems has no ego on the line. Never let the same one grade its own work.
- **CTA:** The full Ramsay agent prompt is what makes it this honest. Comment RAMSAY.
