# raycfu lane — Video Topic Ideas for Alex

> **These are HYPOTHESES.** Each still passes Script Factory Stage-1 kill-gate + a freshness check before scripting. Nothing here is greenlit; these are shots aimed at raycfu's proven sub-lanes, refitted to Alex's Claude-for-builders niche and his 7 HARD RULES.

**How to read each idea:**
Cover-title (Alex's descriptive-noun-phrase style) → spoken first-line hook (raycfu's voice) | sub-lane it extends | Alex-niche fit | breadth check (who can act) | why it's frontier/fresh now | transfer hypothesis (why it overperforms AND survives a faceless mascot reel) | §11 template to fire.

**§11 template legend** (all pull the single Ready-to-Run skeleton from `raycfu-dna.md` §11; the tag names the OPENER variant + the SECTION-LABEL grammar to fire):
- **T-ROLE** = Primary hook ("Most people don't realize…") + named-step grammar ("The [Role].")
- **T-NUM** = Primary hook + numbered grammar ("Prompt 1 / Skill 1 / Step 1")
- **T-CAT** = Primary hook + category grammar ("For marketing, / For developers,")
- **T-CONTRA** = Variant A hook ("Nobody is talking about this, but…") + named-step grammar
- **T-URGENT** = Variant B hook ("You need to pay attention because…") + numbered steps (news-peg lane only)

**Kill-list watch:** flags below mark any idea that drifts toward (a) identity-callout hooks ("If you're a [designer], watch this"), (b) the vague "I went deep and found something nobody's talking about," or (c) a generic product/outcome showcase with no named mechanism. raycfu's real "Nobody is talking about this, but [SPECIFIC SYSTEM]" is SAFE because it lands a concrete claim in the same breath; the kill-list version is the one that stalls on vagueness. Keep the specificity or cut the idea.
