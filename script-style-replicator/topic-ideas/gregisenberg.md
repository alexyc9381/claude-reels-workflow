> HYPOTHESES — each still passes Script Factory Stage-1 + a freshness check.
> Source DNA: `creators/gregisenberg-dna.md`. Niche + rules: `SHARED-CONTEXT.md`.
> 14 ideas, grouped by Greg's sub-lanes. Every idea is frontier (agents / loops / MCP / Skills / councils / Claude Code / autonomous / big-context), outcome-framed, and passes the breadth litmus (a freelance designer, a Shopify owner, a coach, AND a founder could all use it).

**How to read each idea:** Cover-title · first-line hook (Greg's voice) · [sub-lane it extends] · Alex-niche fit · breadth check · why it's frontier/fresh · transfer hypothesis · §11 template to fire (hook variant A/B/C/D + the staircase skeleton).

**§11 hook-variant key** (all fire the same "Start by… Next… But… Finally… → receipt → gated CTA" staircase):
- **A** = "Why would you [use TOOL] only to [waste X]?" (rhetorical provocation)
- **B** = "If you [DO ACTIVITY], Claude just became [NEW ROLE]." (conditional reframe)
- **C** = "Everyone's using [TOOL] like [WRONG MODEL], and wonder why [BAD OUTCOME]." (correction)
- **D** = "You can [use TOOL] to [NUMBER OUTCOME] in [TIMEFRAME]." (number-promise)
- **E** = analogy cold-open ("Imagine a…") · **F** = "X recently dropped Y which is basically [analogy]" · **G** = contrarian thesis
