# The Dopamine Ladder

Framework from **kallawaymarketing** (IG reel, verbatim teach): "6 levels of dopamine released when watching a video. The higher up this ladder you get someone, the more hooked they are." This is NOT new tactics — it is the UNIFYING VOCABULARY over levers we already run. Every rung already has a home in our memories; this file NAMES them so the Script Factory can gate the CLIMB, rung by rung. Load with [[script-factory-pipeline]]; cross-link out, never restate the mechanics.

## The six rungs (L1–L4 are PER-VIDEO; you cannot reach L5/L6 without CONSISTENTLY nailing L1–L4)

- **L1 STIMULATION** — a visual "stun gun" in the **first 2 seconds** (color / motion / brightness / a face) that stops the scroll before a single word registers. Fires here: frame-0-complete-content + front-loaded hero in [[reel-hook-header]]; the real-SCREEN/FACE scroll-stop in [[reel-retention-hook-teardown]]; CLAIM-BEFORE-SPECTACLE (mute-readable claim on screen @ frame 0, 0.0s latency) in [[reel-winning-formula]]. It is NEVER a build-in-progress or a title card.
- **L2 CAPTIVATION** — on top of the stun gun, open a **curiosity loop**: an unanswered question the brain is desperate to close. Fires: the shock-claim opener in [[reel-retention-hook-teardown]]; withhold-the-HOW / redact-the-items / blur-the-prompt in [[reel-hook-header]]; the open-loop / Zeigarnik gap in [[reel-storytelling-playbook]]; the HOOK GATE families in [[claude-reel-hook-library]]. A topic LABEL opens no loop. RECEIPTS-FIRST variant: for a single-number/result payoff, L1 = the result itself on frame 0 and the L2 loop becomes "HOW did it do that?" — front-load the number, withhold only the METHOD.
- **L3 ANTICIPATION** — don't just open the loop, keep them **GUESSING**. The longer you hold the answer without spoiling it, the longer they watch. ⛔ Never pay the promise off mid-video ([[reel-storytelling-playbook]]: close the loop at the END). Fires: the whole [[reel-progress-bar-reward]] anticipation engine (reward seal teased/locked from frame 0), the escalating middle, and GATE THE PAYOFF in [[reel-winning-formula]] + [[gate-the-how-in-scripts]] (make them DESPERATE for the how without ever showing it).
- **L4 VALIDATION** — close the loop and pay it off, ⛔ **BUT THE ANSWER MUST BE NON-OBVIOUS** or the dopamine doesn't release. THIS IS THE NET-NEW RUNG — nothing else states it as a pass/fail. The bar is the [[reel-winning-formula]] INNOVATION test ("make a NORMAL user go 'I didn't know you could do that'"); an obvious/first-guess payoff FAILS. ⛔ GATED-BRAND TWIST (reconciles with [[gate-the-how-in-scripts]]): kallaway closes L4 by GIVING the answer, but our funnel gates the replicable HOW to the comment — so L4 for us = validate the RESULT + name the non-obvious ARTIFACT on-screen (the surprise still lands) while the copy-pasteable method goes to the DM. The keyword ritual is therefore the L4 loop-closer AND the L5 driver at once (value actually delivered in the DM = attachment to the messenger). Scope note: the INNOVATION-BAR runs at **Stage 1 as a TOPIC kill** (is the workflow non-obvious?); L4 is the distinct **PAYOFF-level** check (is the answer to THIS opened loop non-obvious at the moment it closes?). Not the same gate — L4 is genuinely net-new at Stage 4. Pairs with [[specificity-effect]]: a non-obvious payoff stated with an exact figure/name is what makes L4 *feel* real instead of like selling.

## L5/L6 are CROSS-VIDEO brand equity — not per-script scorable, enforced by SERIES CONSISTENCY

- **L5 AFFECTION** — the viewer stops liking just the video and starts liking the MESSENGER.
- **L6 REVELATION** (top) — such a fan that your name/face alone makes them watch the whole thing before they even know the topic.
- ⛔ **FACELESS MAPPING** (this is a faceless Claude/Fable brand, so "the messenger / your face" =): the recurring clay **Claude MASCOT** ([[attack-reel]]) + the byte-identical HOUSE STYLE (cream / Fraunces serif / clay chrome — [[reel-clone-chassis-verbatim]], [[claude-ai-reel-workflow]]) + the **comment-KEYWORD ritual** ([[caption-structure]] + the progress-bar reward unlock). L5 = viewers grow attached to the mascot / house-style / ritual across reels; L6 = those become so recognizable that the BRAND ITSELF is the "face" that pre-sells the watch. The repeatable named SERIES in [[claude-reel-topic-engine]] are the substrate. ⛔ DRIFT KILLS L5/L6 — keep the mascot, palette, serif captions, caption-first keyword and reward seal IDENTICAL every reel; reinventing the chrome resets recognition to zero.

## How to apply (Script Factory)

- **Stage 3 (draft):** the draft must CLIMB all four per-video rungs and the log NAMES the line delivering each via the token `ladder{L1_line, L2_line, L3_line, L4_line, L4_nonobvious:PASS/FAIL}` — L1 (mute-readable stun-gun claim, first breath), L2 (one open loop by ~word 12, a DISTINCT deadline from the value-noun — hitting the value noun does not satisfy L2), L3 (loop held + escalating, never spoiled in seconds 0-5), L4 (non-obvious payoff; gated reel sends the HOW to the comment). A draft missing a rung — no open loop, or an OBVIOUS L4 — FAILS.
- **Stage 6 (storyboard):** hook beat = L1, planted question = L2, escalation curve = L3 (loop held scene-to-scene, obvious-L4 / spoiled-loop = continuity **blocker**, not "major"), CTA payoff = L4 — and EVERY board carries the L5/L6 brand-face invariants (mascot + house style + keyword-reward ritual) UNCHANGED.
- **Stage 4 (gate):** ownership is **SPLIT** so each rung is judged by the critic that can actually observe it (no 7th scorecard dimension), judging as a **NORMAL first-time user, not a Claude insider**. COLD VIEWER (script only) runs TWO passes: (A) tag each of seconds 0-10 in the swipe table with the `rung` it delivers, and (B) a WHOLE-SCRIPT pass for the rungs that land after 0-10s (L3 hold through the middle, L4 payoff at the loop-close). Caps map onto the existing 6-dim scorecard: **L1-claim missing or L2 loop missing = blocker (caps HOOK ≤5)**; **L3 method-spoil or loop paid mid-video = BLOCKER (caps HOOK ≤5, same teeth as gate-the-how)** — but front-loading the RESULT/number is CORRECT for receipts-first and is NEVER an L3 spoil (flag only when the METHOD/answer is given away). The VISUAL half of L1 (motion/color/mascot burst) is NOT judged here — a script has no motion; it is enforced at the render ship-gate (CLAIM-BEFORE-SPECTACLE). RULES LOGGER owns **L4** (end-of-video; reads the Stage-3 `ladder.L4_nonobvious` field + full script): an obvious/expected/first-guess payoff, or a logged L4 FAIL, = **BLOCKER** (never a non-binding "major" — majors don't stop ship under the all-six-≥8 formula, so an obvious payoff would otherwise sail through); an obvious payoff at Stage 4 also means the Stage-1 INNOVATION-BAR was mis-passed → route back.

Pairs with [[reel-scene-motion-depth]] (escalation feeds L1/L3), [[fable-reel]] (reusable open-loop system = L2/L3), and [[specificity-effect]] (specificity is how L4 lands as trust, not selling).
