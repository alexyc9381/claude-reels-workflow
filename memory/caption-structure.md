⛔ Alex has flagged this MORE THAN ONCE ("you keep making this same mistake with the comments"). The written POST CAPTION (the text under the reel, NOT the spoken VO) has a FIXED structure:

**LINE 1 = the comment CTA.** Open with `Comment "[KEYWORD]" and I'll DM you [the named artifact]…` (+ "follow so the DM lands"). The CTA is the FIRST thing, before any value.
**LINE 2 = a "👇 READ BELOW" hook** — e.g. `👇 READ BELOW, this is everything that wouldn't fit in the video:` — that pulls them into the caption body.
**THEN the body/value** — the hook line, the 3 gated workflows/points, the extra context ("stuff that didn't fit"), then close with the CTA again (`Comment "[KEYWORD]"…`) for peak-end.
**THEN hashtags.**

**Why:** the caption's first line is what shows in-feed before "…more", and the whole funnel is comment → auto-DM → guide, so the ASK has to be the very first thing they read. Burying "comment X" after the value = fewer comments. (This is the CAPTION-side mirror of [[reel-winning-formula]]: never open the spoken VO with the CTA, but DO open the caption with it.)

**⛔ ANTI-EXAMPLE (what I keep doing wrong):** writing the caption as hook → value → CTA (CTA in the middle/end). WRONG. The CTA opens the caption.

**✅ CORRECT SHAPE (SOL reel 36, `sol_caption.txt`):**
> Comment "SOL" and I'll DM you the full playbook, the exact copy-paste prompts for all 3 workflows, free. (Follow so the DM lands 📩)
>
> 👇 READ BELOW, this is everything that wouldn't fit in the video:
>
> [hook line] … [1/2/3 gated workflows] … [the part everyone misses] … [a couple things that didn't fit: model-tier cheat-sheet, when-to-use, guardrail principles] … Save this one…
>
> Comment "SOL" and I'll send you the playbook.
>
> #hashtags

Still GATE THE HOW ([[gate-the-how-in-scripts]]): the caption body teaches the what/why/when + shareable context (e.g. the public model-tier/pricing cheat-sheet) but the exact copy-paste PROMPTS stay in the DM'd guide, so there's still a reason to comment. ZERO em/en dashes ([[no-em-dashes-in-copy]]). Pairs with [[lead-magnet-docs]] (the guide the CTA promises).
