Deep-research output (2026-06-18, blended founder/AI-builder lane, spoken-first-line hooks). For Alex's Claude Reels content stream. See [[claude-ai-reel-workflow]].

**6 verified hook families** (survived 3-vote adversarial verification; evidence is practitioner/vendor blogs OpusClip + viral.day + Isenberg's X — DIRECTIONAL patterns to A/B test, not isolated view-count winners):
1. **Hyper-Specific Number/Outcome** — oddly-precise figures read as data, not fluff ("I ran 1,200 prompts… these 3 I still use").
2. **Expert/Proof-First Setup** — credibility → curiosity → payoff in 3 beats. Single most common structure in OpusClip's taxonomy.
3. **Contrarian/Bold Statement** — attacks accepted belief → cognitive dissonance. HIGHEST peak views, LOWEST median (high variance); tanks as ragebait without substance.
4. **Open-Loop Question** — specific, un-answerable yes/no → information void.
5. **Pure Curiosity-Gap / "feels illegal to know"** — the ONE format verifiably viral in the wild (Threads/X). Must pay off the loop.
6. **Direct Promise** — measurable outcome + number beats vague tease.

**KILLED in verification (do NOT use):** "If you [identity], watch this" identity-callout (0-3); "Investigator — I went deep and found something nobody's talking about" (0-3); "Product/Outcome Showcase = highest performer" (0-3); most "X% watch past 3s" stats (fabricated).

**AUDIENCE GUARDRAILS (critical for Alex's founder audience):** (a) NO scammy income/follower flexes ("$43→$2,109", "0 to 50k followers") — reads get-rich-quick. Use DOMAIN-RELEVANT proof shown ON SCREEN (real Claude output, ship logs, before/after). (b) Peer-to-peer, never beginner-explainer. (c) Every curiosity/question hook MUST deliver — 2026 algos punish unpaid loops.

**PROCESS (Isenberg, verbatim from X):** "before opening any tool, write 10 first lines that would stop someone mid-scroll. emotion, surprise, or curiosity." → draft 10+ first lines per video, record/test best 2-3. Re-test quarterly (hook fashions shift fast).

(contrarian, his Loops-reel angle); "There's a way to use Claude that honestly feels illegal to know about." (curiosity-gap). **HOOK GATE — run EVERY hook through this before recording (consolidated 2026-06-20; the standing process Alex asked to enforce on ALL hooks, this reel + future).** Every hook MUST clear all 10:
1. **Volume first** — draft 10+ first lines before opening any tool (Isenberg); record/test the top 2-3 across **≥2 different families** (different risk profiles: a low-variance numeric promise + a high-variance contrarian + a curiosity/tripwire).
2. **Punch in sentence ONE** — no warm-up/setup line before the claim; the first sentence IS the hook (Alex's rule). Bad: "Your competitors don't stop working when you go to bed. So why does your Claude?" (punch is in sentence 2) → Good: "Claude can do eight hours of work while you sleep — and have it on one page by 7am."
3. **Mute-readable** — a NUMBER or a hard CONTRAST burned on frame 1, and the opening VISUAL carries the meaning sound-off (title+thumbnail test).
4. **Specificity = the #1 lever** — oddly-precise > round numbers; bait the STAKE (their money / time / decision / blind-spot), NEVER the tool ("how to prompt Claude" is a Core-only ceiling that won't travel). A hook with no number/contrast usually FAILS this gate — add one.
5. **Use a verified family** (hyper-specific number / proof-first / contrarian / open-loop question / curiosity-gap / direct-promise). AVOID the KILLED patterns: identity-callout ("if you're a founder, watch this"), "I went deep and found something nobody's talking about," product/outcome showcase, fabricated "% watch past 3s" stats.
6. **Self-select where possible** — stage + symptom + this-week timing in the first ~5 words ("if you wake up already behind — a hundred tabs, an inbox you're scared of…") beats a generic "you."
7. **Guardrails** — peer-to-peer, never beginner-explainer; NO scammy income/follower flexes; show domain-relevant proof ON screen (real Claude output / ship logs / before-after).
8. **Pay the loop** — every question/curiosity/open-loop hook MUST be paid off later in the reel (2026 algos punish unpaid loops).
9. **Sound human** — no AI vocab/structural tells; dinner-table test; ≤1 short staccato punch in a row (see [[shortform-scripting-playbook]] humanize section).
10. **Freshness** — never reuse a prior video's opening LINE or opening VISUAL; re-test families quarterly (hook fashions shift fast).
