**Pause-tightening method (generalizes dead-air/cut-cut handling, [[caption-sync-gate]] + [[video-editing-toolchain]]):** to tighten a recorded VO in ONE pass at full quality, run on the original m4a: `silenceremove=start_periods=1:start_silence=0.05:start_threshold=-40dB:stop_periods=-1:stop_silence=0.4:stop_threshold=-40dB` (trims lead + caps EVERY pause to 0.4s) then the polish chain, `-ar 48000`. ⚠️ This changes timing, so you MUST re-transcribe the output — and capping pauses can EXPOSE "cut cut" retakes the first whisper pass hid (reel 16 had two hidden cut-cuts on the "actual premium on-brand graphics" line that only showed after tightening). Always splice the retakes (keep the clean take), re-transcribe, confirm `cut` count = 0 before building words/L.
