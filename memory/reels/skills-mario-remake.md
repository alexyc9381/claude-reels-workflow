## ⭐ REUSABLE: full-theme reel remake method
1. **Foundation first, verify, THEN scenes.** Add the theme to the two SHARED things — the `Bg` (Mario sky+clouds+hills+brick ground) and a Mascot COSTUME prop (`mario?:number` → red cap w/ C, mustache, overalls) — render one still to confirm, before touching any scene body. Costume slots into the existing Mascot the same way glasses/wizard/judge do.
2. **Build a small theme KIT of reusable prop components** (MSky, QBlock, Brick, Coin, Pipe, Goomba, Star, FireFlower, Mushroom, HudBox, MSlug) at module scope, then each scene composes from them. Keeps scenes short + consistent + declutter-friendly.
3. **⛔ absolute-positioned prop in a flex row = all stack at one x.** QBlock/Coin/Star/etc are `position:absolute` with an `x` prop — putting them in a `display:flex` container makes flex ignore them and they pile on top of each other (looked like ONE coin). Give each an explicit `x={i*step}` inside a plain positioned box, never rely on flex gap. (Bit R0 power-ups + R3 coin row.)
4. **Zone every scene into non-intersecting rectangles** (the recurring [[reel-no-emoji-no-lowopacity]] rule #4). Fixed R0 (mascot jump peak crossed the big number → split to left-action / right-payoff columns), R2 (SAVED banner sat on the flag+mascot → moved below), R6 (SKILLS stamp/cards ran under the castle → center lane between flag x≤210 and castle x≥712), R3 (stats text hidden behind mascot → mascot own bottom-left lane).
5. **Transition-out must fully clear, not linger at low opacity.** R3's "before" panel fading to `opacity:0.15` and sitting there ~6s read as an unfinished ghost (violates no-low-opacity). Slide it out + `Math.max(0,1-transform*1.5)` so it's gone.

## ⭐ REUSABLE: original CHIPTUNE game SFX via numpy (no copyright, matches theme)
Synthesize the game's SOUND SHAPES as original square-wave waveforms — never Nintendo samples. Generator persisted at `matchtern-longform/video/tools/gen_mario_sfx.py` → writes `public/sfx/m_*.wav`: coin (two-tone B5→E6 square), bump, jump (rising square sweep), powerup (ascending arpeggio), 1up (6-note jingle), stomp (noise+low square), pipe/warp (descending glissando), flag (fast down-arpeggio), clear (triumphant fanfare + low harmony). Env = exp/linear decay, `np.tanh` soft-clip, peak ~0.72. **numpy 2.0.2 is on the system python3; scipy is NOT** — use only numpy + stdlib `wave`. LAYER these at theme beats (v ~0.2-0.34) ON TOP of the existing cinematic riser/booms (don't replace them — keep the hook + CTA risers Alex asked for). Verify no clip: ffmpeg `volumedetect` max_volume < 0 dB (got -2.2). Same method reusable for any themed reel (arcade, RPG level-up, etc).
