# ⛔⛔ OPEN THE FACTORY LOG FIRST. IT IS NOT PAPERWORK — IT IS THE PIPELINE.

> Alex, 2026-07-15: *"WHY DIDNT U FOLLOW MY ORIGINAL WORKFLOW JUST MAKE SURE YOU FOLLOW THE CORRECT
> WORKFLOWS ... LIKE WITH THE MD AND STUFF ... YOU HAVE TO MAKE SURE YOU DO THAT IN THE FUTURE"*

**Create `claude-reels-workflow/memory/reels/<keyword>-factory-log.md` the moment a premise is named — BEFORE
pitching it, BEFORE writing a hook, BEFORE any script.** Then fill it stage by stage as each one *completes*.
The playbook already said *"No complete factory log = no recording"* and *"unlogged enforcement = no gate."*
**The log is not a record of the work. Writing it IS the work** — it is what forces Stage 0 to have a comp,
Stage 1 to log 10 rules with evidence, and Stage 4 to happen before a storyboard is offered.

⛔ **NEVER write it retroactively to justify a decision already made.** A log authored after the pitch is a
press release. Reel 59's was opened at Stage 4 and every skipped stage above it had to be reconstructed —
which is exactly how the skipping went unnoticed for a whole session.

## The order. It is not negotiable and it is not re-orderable.
```
STAGE 0  SOURCE        → door + comp URL + views + multiple + age + transfer hypothesis.  ⛔ No comp = no entry.
STAGE 1  KILL-GATE     → all 10 rules, pass/fail, WITH EVIDENCE. ⛔ Harsh audience binding, no conditional penalty.
STAGE 2  STRUCTURE     → structure_comp beat map + template. ⛔ Single dark panel, never dual-screen.
STAGE 3  DRAFT         → 10+ hooks across ≥2 families, top 3, the VO.
STAGE 4  GATE A        → 3 fresh critics. SHIP = all six ≥8 + zero blockers. ⛔ NOT GATED = NOT RECORDED.
STAGE 4.5 CAPTURE      → ⛔ CAPTURE FIRST, SCRIPT SECOND. Every claim gets a real shot or it is cut.
STAGE 6  STORYBOARD    → one card per scene + the reasoning. ⛔ No approved board = no build.
STAGE 7  BUILD         → wireframe → overhaul → SFX → ship.
```
⛔ **You may not offer stage N+1 until stage N is logged.** Offering a storyboard on an ungated script is the
same class of error as FOREMAN's gate grading a hook the reel didn't use.
