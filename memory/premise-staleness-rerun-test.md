# ⛔ RULE 10 — THE RERUN TEST (external staleness)

Added 2026-07-15 after **Alex killed reel 59 SKETCH on instinct** ("i feel like the sketch idea is outdated")
that three rounds of a 3-critic gate could not kill. Runs at **kill-gate time** ([[vault-reel-premise-autopsy]] /
CLAUDE-REELS-PLAYBOOK §A2), alongside the other 9.

## The test

**Name the year the audience first saw this exact REVEAL — from anyone, not from us.**
Older than **~12 months** = the hook's "no way" is a lie the viewer already caught. **No rewrite fixes it. KILL.**

This is **external** collision. The existing rules only measure **internal** collision:
- rule 7 lever-rotation → against [[reel-lever-ledger]]
- A3 arena's growth-strategist "novelty vs shipped" → against *our* 58 reels

Neither one asks the only question that matters: **has the feed already shown this demo?**

## ⛔ Why it MUST die here

**GATE A physically cannot catch it.** Its six dimensions are gate-integrity · concreteness · hook · believability ·
topic-breadth · structure-fidelity. **There is no novelty axis** — the scorecard has no box to put it in. A premise
can be a decade old and score 8s across the board.

## The tell: a freshness note that passes VIA A CAVEAT

> *"genuinely frontier **and rarely shown end-to-end**"* · *"everyone's seen X **but nobody does it with** Y"*

The caveat is doing all the work → the premise is old. **Then ask where the wow rides:**
- wow on the stale half + caveat on an afterthought beat → **KILL**
- rebuild so **the WOW *is* the caveat** → that is a *different premise*: re-enter at A1 with its own comp

⛔ Never let the idea's own author grade its freshness. `script-style-replicator/topic-ideas/*.md` self-graded
SKETCH "genuinely frontier." That is marking your own homework.

## ⭐ The diagnostic lesson (the transferable one)

GATE A rounds 1-3 kept reporting **the same wound in different words** — *"no open loop by word 12"*, *"the header
already answered the only thing 'No way' could point at"*, *"gate scope inverted"* — and I kept patching **wording**
(planting loops, moving the gate, re-scoping the CTA).

> **The critics were describing the symptom. The premise was the disease.**
> **When a gate fails the same beat 3 rounds running on different wording each time, STOP PATCHING AND RE-OPEN THE PREMISE.**

A hook with "nothing left to reveal" is usually not a badly written hook. It is a **reveal the audience already owns**.

## Worked kill — reel 59 SKETCH

"Photo of a napkin sketch → a real working app." Sketch→app was **GPT-4V's March 2023 launch demo**, then tldraw
"make it real" (Nov 2023) — arguably the most-replicated AI demo ever. The fresh half was the **deploy** (→ a live
link strangers open); the topic file put the weight on the hook, i.e. **the sketch**, i.e. the 3-year-old half.
⛔ Do not resurrect the sketch angle: the sketch is simultaneously the only distinct thing in it and the only stale
thing in it. Strip it and it's a generic vibe-code deploy reel (= SWIPE-50's lane).
