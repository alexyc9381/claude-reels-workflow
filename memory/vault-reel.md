Fixes that came from the adversarial critic gate (all real, recurring):
- **Gate on-screen "rule" text**: the reframe first PRINTED the real rules ("weigh the downside first"…) = ungated the payoff. Fix = meta-labels + REDACTED (blurred) value blocks + 🔒 per row ("your downside rule ▓▓▓ 🔒"). Prove the vault EXISTS, never show its content.
- **Badge must match the beat**: a persistent "FABLE 5 · 3 DAYS FREE" chip on the PAID/after scenes contradicts "now paid". Added a `paid` prop → red "FABLE 5 · NOW PAID" on scenes 3-4.
- **Countdown number must match the headline word**: big "02 DAYS" vs headline "3 more days" reads as a bug on mute → set clock to "03 DAYS".
- **Sibling sets must settle together**: the 3 cheap-model reader pills staggered so a hold frame showed one missing + greyed = looks broken + contradicts "every cheap model". Floor pill scale (never 0), tighten stagger so all settle ~1s earlier, reduce grayscale.
- **Progress-bar reward node**: the traveling avatar reached 100% and swallowed the end gift node. Clamp the traveling Mascot `left` to `Math.min(p,0.9)*100%` so the gift stays a distinct node.
- **Caption chunker**: never end a caption line on a dangling word (i/a/the/to/of/and/is/it/you) — defer the break one word.
