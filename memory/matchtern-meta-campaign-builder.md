Working Meta Marketing API campaign builder lives at `~/Downloads/meta-campaign/`:
- `build-campaign.mjs` ‚Äî zero-dependency Node (uses native fetch/Buffer; machine has Node v24). Creates **campaign ‚Üí ad set ‚Üí ads, ALL `status: PAUSED`**. Dry-run by default; `--execute` to create; `--execute --delete <CAMPAIGN_ID>` to tear down a test build. Writes created IDs to `last-build.json`.
- `campaign.config.json` ‚Äî editable plan: campaign (OUTCOME_LEADS, $50/day campaign budget), one ad set (Parents, US, 30‚Äì60, parent-interest stack, Advantage+ audience+placements), and 6 ads mapped to `ad-1..ad-6` PNGs with primary text/headline/description (copy adapted to a Learn More ‚Üí landing-page CTA, NOT the organic "Comment INTERN").
- `.env.example` ‚Üí copy to `.env`: `META_ACCESS_TOKEN` (needs `ads_management`), `META_AD_ACCOUNT_ID` (digits, no `act_`), `META_PAGE_ID`, optional `META_PIXEL_ID`, optional `META_IG_ACCOUNT_ID`.
- `README.md` ‚Äî full setup/run/troubleshoot.

**Hard boundary (keep):** the script never turns ads on, never sets budget live, never touches a payment method, never publishes. Alex flips it live in Ads Manager himself. Don't add a publish/activate path. Token is read locally from `.env` and sent only to Meta ‚Äî never put it in chat or commit it.

**API notes that work:** images upload via `POST act_<id>/adimages` with base64 `bytes` (grab `Object.values(res.images)[0].hash`); interests resolved by name via `GET /search?type=adinterest`; spec fields (targeting, object_story_spec, promoted_object, call_to_action) must be JSON-stringified in the urlencoded POST body; `OUTCOME_LEADS` + `OFFSITE_CONVERSIONS` needs `promoted_object={pixel_id,custom_event_type:LEAD}` ‚Äî script auto-falls back to `LINK_CLICKS` if no pixel; `special_ad_categories` ‚Äî **OPEN gray-area call** (Alex is sitting with it, leaning "program, not placement"). If the ad/LP is **placement-framed** ‚Üí `["EMPLOYMENT"]` (career placement is in Meta's Employment scope) + targeting refit (no age/gender/ZIP/lookalike); if **genuine-program/admissions-framed** ‚Üí likely `[]` (NONE). Determination follows what the ad says, not intent. Config still ships `[]` pending Alex's decision ‚Äî see [[fb-ads-compliance-skill]].
