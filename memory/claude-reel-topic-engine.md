**BREADTH RULE (added 2026-06-19 — Alex's correction):** topics must CAST A WIDE NET, not deep niche ops. Alex flagged a batch of deep B2B-SaaS-founder plays (security questionnaires, Stripe reconciliation, PR review gates, funnel swarms, onboarding experiments) as "too deep" — each only ~10% of the audience could use it. **Litmus test: could a freelance designer, a Shopify store owner, a coach, AND a startup founder ALL immediately use this?** If only a technical founder can, it's too deep. Audience is broader than "founders who know Claude cold" — it's anyone running or building anything (solopreneurs, freelancers, agencies, small biz, ecommerce, creators, coaches, consultants, founders). BUT still NOT generic/beginner/consumer fluff ("summarize a PDF", "write my emails") — the breadth is in WHO can use it, the value bar stays high (a smart, specific, screenshot-worthy play). Aim topics at: make more money / save real hours / get more customers / look pro / smarter decisions, in ways that apply broadly.

**5 REPEATABLE SERIES (durable formats that never run out):**
1. **While You Slept** — queue ONE real task to Claude Code before bed (backlog, 50 cold emails, rebuild a page), film the 7am reveal with a live counter + an honest "what it got wrong / where I gated it" beat. Runs forever: infinite backlog × finite night, new function every episode. Rides the hottest Q2-2026 format (Alex Finn "ClawdBot while I slept" / Jack Clark "parallel worlds").
2. **200 Agents, One Problem** — hand Claude one hard problem with a scoreable target, unleash a subagent swarm (Dynamic Workflows), pure split-grid motion-graphic spectacle with an eval counter climbing to convergence, no talking head. Any scoreable founder problem = a fresh episode; brand-new May-2026 capability = pre-saturation/frontier.
3. **Steal This Founder's Claude Stack** — faceless screen-recording teardown of one operator's real setup (CLAUDE.md, subagent configs, MCP, slash commands), captions carry it, ends on a copyable artifact. Often a contrarian hook ("you're using CLAUDE.md wrong"). Unlimited founders × constantly-shipping primitives; swipeable configs drive saves.
4. **I Gave Claude My P&L** — feed it something founders never hand an AI (real financials, churn export, pipeline, ad account); outcome-first "receipts" rhythm: open on the number/finding, reverse into the prompt. Every company's numbers differ and change monthly = endless; "receipts over demos" = highest-trust angle.
5. **Claude vs a $10K Agency** — head-to-head vs a job founders outsource (brand, 6-email flow, SEO cluster, landing page); on-screen scorecard on cost/turnaround/quality. Endless agency-deliverable menu; versus-frame = built-in tension; carries Isenberg's "30 engineers is now 1 person + a Claude sub" thesis.
